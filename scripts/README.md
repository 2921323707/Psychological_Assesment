# 数据库初始化脚本

## 说明

本目录包含数据库初始化和数据导入脚本。

## 使用方法

### 初始化量表数据

运行以下命令来初始化数据库并导入量表数据：

```bash
python scripts/init_scales.py
```

### 功能说明

- 自动删除现有数据库表并重新创建（包含最新的字段结构）
- 导入以下量表：
  - **SDS（抑郁自评量表）** - 20题
  - **SAS（焦虑自评量表）** - 20题

### 注意事项

⚠️ **警告**：运行此脚本会删除所有现有数据！

如果需要保留现有数据，请：
1. 先备份数据库文件 `psychological.db`
2. 或者修改脚本，使用数据库迁移工具（Flask-Migrate）来更新表结构

### 数据结构

每个量表包含：
- 基本信息：名称、描述、分类、题目数量
- 题目数据：问题、选项、反向计分标记、排序

### 扩展更多量表

要添加更多量表，请编辑 `scripts/init_scales.py`：

1. 添加量表题目数据（格式参考 `SDS_QUESTIONS` 和 `SAS_QUESTIONS`）
2. 在 `init_scales()` 函数中添加创建逻辑
3. 运行脚本重新初始化

---

**注意**：本脚本主要用于开发环境。生产环境建议使用 Flask-Migrate 进行数据库迁移。
